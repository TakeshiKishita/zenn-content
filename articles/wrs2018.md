# WRS2018に出場しました


10月の話ですが、半年ほど準備をしていたWRSに出場してきました。  

賞金総額1億、カテゴリ優勝1000万『世界のロボットの叡智を集めて開催する競演会』(白目)に、  

素人チームがなぜか審査が通っちゃって出場したので、レポートします。


# 参加のキッカケ


機械学習の勉強で何か作りたいなと考えていたところ、ちょうど友達に自動運転やろうと誘われたので、少し作っててみることにしました。  

自分はシステム担当で、白線をとりあえず白線を追従するラジコンでも、と思ってこんなのを作ってました。  

![lane_detect](https://forum.avinton.com/uploads/default/original/1X/0ed350bb438e9035b31598e267abf4f46ebdab58.gif)


が、マシン担当の友人が結局何も作らなかったので、自分で車体も組んでみることに。  

車体を組む段階で、車体をコントロールするためのモータードライバ、Raspberry Pi、Arduino、GPIO等よくわからなかったので、もくもく会に潜入したところ、なぜかそのままの流れで大会出場まで行ってしまいました。


# WRSとは


2020年のWorld Robot Summit（WRS）本大会の開催を前に、今年2018年は東京大会を開催します。  

WRSは、ロボットの活躍が期待されるさまざまな分野において、世界中から集結したチームがロボットの技術やアイディアを競う競技会「World Robot Challenge（WRC）」と、ロボット活用の現在と未来の姿を発信する展示会「World Robot Expo（WRE）」とで構成されています。  

ロボットをテーマに人々がつながり、未来を語り合う場。WRSへの皆様のご来場を心よりお待ちしています。


つまり、2020年に本大会があり、そのプレ大会が今大会でした。

# ハプニング！


その場での対応能力も、大会で大事な要素です、、、が、  

全員大会出場経験なんて無いので、普通なら予測できたトラブル等も色々やらかして、  

大会中ずっとトラブってました。


* マシン動かない（無線接続が混線により全て切断される）
* 故障する（ネジ取れたり、足が変な方向向いたり）
* アームが動かない（もうまぢむり）
* 初日に動かないアームを乗せずに出場したところ、その後の換装はルール違反のためアームを消失  

（ルールを確認してないリーダーに、チームの雰囲気は最高潮へ）


# 学び


今回の大会に出て一番勉強になったことは、


* **物理の開発は超たいへん！**


	+ モノが無いとそもそも開発できない。リモートで参加とか、  
	
	平日に家で作っておくとか出来ないのが辛い。
	+ 機械とか、パーツとか、モーターとか普通に壊れる→イチから作り直し、買い直し。
	+ 中華製は普通に動かない！
	+ dockerコマンドで全てを無かった事にできるシステム開発ってサイコー！
	+ 「お金かかる！」システム開発は無料でいろんなリソースが提供されてるけど、  
	
	マシン開発はネジ1本から全部買わないといけない。  
	
	予備とか欲しいけど高いし買えない→壊れる→中国から3週間後に届く
* **チームビルディング、リーダーの選定は大事**  

蓋を開けてみたらチーム5人、実質エンジニア2人みたいな感じでした。  

大会運営とのパイプ約はもちろんリーダー、しかし、コード書かないし、口だけ達者。  

大会にエントリーして色々お金かけてるし、ここまで作ったら、引くに引けず。。。  

何か共同で制作をする際は、チームビルディング、メンバーが大切だと学びました。




# 大会を通して


![44706964744_a6a0cd8953_k](https://forum.avinton.com/uploads/default/original/1X/3a7c00f50fda6db9b7093d56a271cfadef773dcf.jpeg)  

エンジニアの大会に始めて出て、大会独特のライブ感は、辛くも楽しい経験でした。  

半年くらい準備して望みましたが、全然足りません。  

ガチの大学と企業が鎬を削ってる中で、素人が週末半年だけじゃ全然敵いませんでした。  

2年後出ますか？と、大会後の主催者アンケートで聞かれましたが、即答できないくらいには難しい世界でした。  

2年後出るなら今から準備しないと・・・という感じです。


ちなみに、今大会の優勝は東大＋企業のチーム、2,3位はドイツチームで、  

私のチームはギリギリ下から2位でした。


![DSC_0090](https://forum.avinton.com/uploads/default/original/1X/5301a2331ed6cd8d41281911de8278c7861fff66.jpeg)  

2位のドイツチームのロボット、フルオプションにするとBMWと同じ位のお値段だそうです。。。


上にも書きましたが、今回の一番の学びはチームビルディング。  

技術云々の前に、チームが出来上がらないとヤル気も成績も出ません。  

これは仕事にも言えるますが、プライベートでもせっかく自分の時間を割いているので、  

いい環境で開発したいものです。  

担ぐ人はくれぐれも厳選しましょう。


機械学習エンジニアになったと思ったら、今までほとんど機械学習と無縁な週末を送っていました。  

ただ、ロボットにも興味はあったし、学びも多く、普通出れないようなクオリティの大会にも出れたので良しとします。


制限された時間の中で、周りの人競ったり、トラブルシュートするのはすごく楽しかったです。  

皆さんも大会、コンテストの出場オススメです！（ロボットはあんまりオススメしないｗ）
